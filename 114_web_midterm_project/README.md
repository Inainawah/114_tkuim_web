# UniLife Toolkit｜課程規劃 + GPA 即時計算 + 番茄鐘

## 組員
- 曾彥勳（412630518）
- 黃柏誠（412631052）

## 專案簡介
一個整合「課程規劃（含即時 GPA/加權平均）」與「番茄鐘＋任務清單」的校園效率工具。支援深色模式、RWD、localStorage 永續化。

## 使用技術
- HTML5 語意化結構（header/nav/main/section/table/aria）
- CSS3、Bootstrap 5（RWD、排版、元件）
- JavaScript（querySelector、addEventListener、createElement、localStorage）
- 表單驗證：HTML5 Constraint Validation API + 自訂錯誤訊息
- GitHub Pages 發佈

## 功能特色
1. 課程表單驗證（名稱、學分、成績格式）與防重複輸入  
2. 即時計算：總學分、加權平均（百分制）、GPA（4.3 制）  
3. 課程搜尋、刪除、CSV 匯出  
4. 番茄鐘（自訂時長、4 次專注自動長休息、完成數統計）  
5. 任務清單（新增/完成/刪除/清除已完成）  
6. 深色模式、動畫過場、完全 RWD  
7. 資料持久化（localStorage）
