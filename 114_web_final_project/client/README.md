# React + Vite 運動追蹤管理系統 (前端)

這個專案是使用 React 和 Vite 建構的運動追蹤管理系統前端介面。

## 功能特色
- **新增運動記錄**：輸入運動名稱、重量、次數、時間與距離。
- **檢視列表**：即時顯示所有運動記錄。
- **刪除功能**：快速刪除不需要的記錄。
- **響應式設計**：使用 Tailwind CSS，支援各種裝置瀏覽。

## 專案設定

### 1. 安裝依賴
```bash
npm install
```

### 2. 啟動開發伺服器
```bash
npm run dev
```
前端預設運作於 `http://localhost:5173`。

## 技術堆疊
- **框架**: [React](https://reactjs.org/)
- **建置工具**: [Vite](https://vitejs.dev/)
- **樣式**: [Tailwind CSS](https://tailwindcss.com/)
- **路由**: [React Router](https://reactrouter.com/)
- **HTTP 客戶端**: [Axios](https://axios-http.com/)
