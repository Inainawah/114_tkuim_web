# 米哈遊抽卡分析計算機
本工具使用 **蒙地卡羅模擬法 (Monte Carlo Simulation)**，通過執行 10,000 次模擬抽卡，來預測在特定抽數下獲得五星角色或是武器的期望值與機率。
- **雙模式支援**：
  - **角色活動祈願**：模擬 90 抽保底、50/50 機率歪卡機制、大保底機制。
  - **武器活動祈願**：模擬 80 抽保底、75/25 機率歪卡機制、大保底機制。
- **視覺化介面**：
  - 精美的星空背景動態效果。
  - 響應式卡片設計。
- **即時分析數據**：
  - 計算平均獲得的限定五星數量。
  - 計算平均獲得的五星總數。
  - 分析「至少獲得一隻限定五星」的成功機率。

1. **開啟網頁**：直接使用瀏覽器打開 `index.html` 檔案。
2. **選擇卡池**：點擊上方的切換按鈕，選擇「角色活動祈願」或「武器活動祈願」。
3. **輸入抽數**：在輸入框中填寫你目前擁有的抽數（例如：160 抽）。
4. **開始分析**：點擊「開始分析」按鈕。
5. **查看結果**：系統將自動執行 10,000 次模擬，並在下方顯示統計結果：
    - **期望限定五星**：平均能拿到的當期 UP 角色/武器數量。
    - **期望五星總數**：平均能拿到的所有五星（含歪常駐）數量。
    - **至少獲得一隻限定的機率**：你這次抽卡能「畢業」的機率。

## 檔案結構

- `index.html`: 主頁面結構。
- `style.css`: 樣式表。
- `script.js`: 抽卡模擬演算法與 UI 邏輯。
